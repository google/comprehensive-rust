# Moved Strings in Rust

<!-- mdbook-xgettext: skip -->
```rust,editable
fn main() {
    let s1: String = String::from("Rust");
    let s2: String = s1;
}
```

* The heap data from `s1` is reused for `s2`.
* When `s1` goes out of scope, nothing happens (it has been moved from).
