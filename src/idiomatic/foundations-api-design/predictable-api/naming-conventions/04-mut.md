# Mut

Suffix for access-style methods.

```rust
Vec::get_mut
// Iterate over a slice by mutable reference, being able to mutate
// items in the iterator as you iterate over them.
slice::iter_mut
RefCell::get_mut
```

<details>
- Mut for Mutability

- Suffix that signifies the method gives access to a mutable reference.

  Requires mutable access to the value you're calling this method on.

</details>
